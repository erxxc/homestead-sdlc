# Privacy Notice — Homestead SMP

## Data Controller
Homestead SMP — operated by the server administrator.

## Data We Collect

| Data Type | Purpose | Retention |
|---|---|---|
| Minecraft UUID | Player identification, ban/whitelist management | Indefinitely while player is active |
| IP address | Security logging, fail2ban, connection logs | 30 days in server logs |
| Playtime statistics | Server metrics, Grafana dashboard | 90 days in Prometheus |
| In-game actions | Security audit trail | 30 days in audit log |
| Chat messages | Server logs | 7 days in latest.log |

## Lawful Basis
Legitimate interest — operating a multiplayer game server requires processing
player connection data to function correctly and maintain security.

## Data Sharing
- Player data is not sold or shared with third parties
- Aggregated anonymous statistics may appear on the public status site
- Player names are never displayed publicly without consent

## Player Rights
Players may request:
- Access to data held about them
- Deletion of their data (note: some data required for security purposes)
- Correction of inaccurate data

Contact the server administrator via Discord to exercise these rights.

## Security
Player data is protected by:
- Encrypted server connections (online-mode=true)
- Access controls on server infrastructure
- Regular backups with restricted access
- Audit logging of administrative actions

## Changes
This notice may be updated. Players will be notified of significant changes
via the Discord server.

## Contact
Server administrator contact available via the Discord server.
