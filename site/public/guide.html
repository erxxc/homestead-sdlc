Getting Started

Everything you need to join the server in under 10 minutes.

Requirements

Minecraft Java Edition1.20.1
LauncherCurseForge App (recommended) or Modrinth
RAM allocated6GB minimum, 8GB recommended
Modpack versionHomestead 1.3.4

Installation

1
Download and install the CurseForge app
Go to curseforge.com and download the CurseForge app for your operating system. Open and log in with your Minecraft account.
2
Find Homestead
Click Browse Modpacks, search for "Homestead Cozy" and select the pack by Toekimi. Make sure to select version 1.3.4.
3
Install and allocate RAM
Click Install and wait for all mods to download. Once installed, click the three dots next to the Play button → Profile Options → set Memory to at least 6144 MB (6GB).
4
Launch and join
Click Play. Once in the main menu go to Multiplayer → Add Server and enter the server address below.
1
Download and install the Modrinth app
Go to modrinth.com/app and download the Modrinth launcher for your operating system.
2
Find Homestead
Click Browse, search for "Homestead" and select the pack by Toekimi. Select version 1.3.4 from the version dropdown.
3
Install and allocate RAM
Click Install. Once complete, click the three dots on the instance → Edit → Java Settings → set Maximum Memory to at least 6144 MB.
4
Launch and join
Click Play. Once in the main menu go to Multiplayer → Add Server and enter the server address below.

Performance tip

If you experience low FPS, disable shaders in Video Settings → Shader Packs → Disable. Homestead runs well without shaders on most systems.

Voice Chat

Homestead includes Simple Voice Chat. Once in-game press V to open voice chat settings. Make sure your microphone is selected and set to push-to-talk or open mic based on your preference.

Having trouble?

Join our Discord and post in #community-support. Include your launcher, modpack version, and a description of the issue.
